***README*******
Assumption:
This readme assumes that the user is a proficient user of the Azure environment
To install Pandas on Linux vm on Azure:
1. ssh to linux vm using password or private key

2. Run the command below to update the linux environment
sudo apt-get update -y" 

3. Run the command to install python3
sudo apt-get install python3-distutils \
sudo apt-get install python3-apt

4. Run the command below to install pip
sudo apt install python3-pip

5. Run the command below to install pandas
sudo pip install pandas

To install Mage AI on linux vm :

Run the following command:
sudo pip install mage-ai
To allo inbound traffic to port 6789:

# Install Google Cloud Library
sudo pip3 install google-cloud

sudo pip3 install google-cloud-bigquery
